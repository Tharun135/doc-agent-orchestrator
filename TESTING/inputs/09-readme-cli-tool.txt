# Installation

Install the CLI using npm:

```bash
npm install -g our-cli-tool
```

## Configuration

Create a config file in your home directory. The tool will use default settings if config is missing.

## Usage

Basic command structure:
```
our-cli [command] [options]
```

Use `--help` flag for detailed information about any command.

## Authentication

The tool requires API credentials. Set them as environment variables or pass them via flags.
