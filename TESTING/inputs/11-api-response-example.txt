## Test Input 11: API Response Example (Documentation Need)

Example successful response from /api/users endpoint:

```json
{
  "status": "success",
  "data": {
    "user_id": 12345,
    "username": "john_doe",
    "email": "john@example.com",
    "role": "admin",
    "created_at": "2024-01-01T10:00:00Z",
    "last_login": "2024-01-09T08:30:00Z",
    "preferences": {
      "theme": "dark",
      "notifications": true
    }
  }
}
```

Error response when user not found:
```json
{
  "status": "error",
  "error_code": "USER_NOT_FOUND",
  "message": "User does not exist"
}
```
