# Data Sync

## Overview
The data sync feature keeps local and remote data in sync. Run the sync job to update data. Sync can be run manually or on a schedule.

## Setup
1. Configure the sync settings.
2. Set the sync interval.
3. Enter credentials.
4. Test the connection.
5. Enable sync.

## Running a Sync
1. Trigger sync from the appropriate location.
2. Wait for sync to complete.
3. Check the results match expected values.
4. If errors occur, check logs and retry.

## Notes
- Large datasets may take longer to sync.
- Sync runs according to standard schedule by default.
- Refer to the configuration guide for advanced options.
