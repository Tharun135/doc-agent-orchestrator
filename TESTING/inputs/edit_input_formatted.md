# Editing an Existing Data Source

## Prerequisites

- Access to the IEC 60870-5-104 connector
- An existing data source to edit

## Procedure

1. Go to **Get Data**.
2. The available connectors are displayed.
3. Select **IEC 60870-5-104**.
4. The overview opens.
5. Select the data source from the list.
6. Navigate to **Tags**.
7. Edit the data source.
8. Change the configuration.
9. Save the changes.

## Notes

This procedure applies to editing existing IEC 60870-5-104 data sources.
