## How to Create These Assets:

### For Screenshots:
1. Use Windows Snipping Tool / macOS Screenshot tool
2. Or use tools like:
   - ShareX (Windows)
   - Monosnap (Mac/Windows)
   - Lightshot (Cross-platform)

### For GIFs:
1. Record with:
   - **LICEcap** (Free, cross-platform)
   - **ScreenToGif** (Windows, free)
   - **Kap** (Mac, free)
   - **Peek** (Linux, free)

2. Settings:
   - Frame rate: 10-15 fps
   - Resolution: 1280x720 or 1920x1080
   - File size: Keep under 5MB per GIF

### For Diagrams:
1. Use:
   - **Excalidraw** (https://excalidraw.com) - Free, simple
   - **Figma** (Free tier available)
   - **draw.io** (Free)
   - **Canva** (Free tier)

## File Naming Convention:

- Use lowercase with hyphens
- Be descriptive
- Match the names used in README-NEW.md

## Image Specifications:

- **Format:** PNG for screenshots, GIF for animations
- **Max width:** 1000px (will auto-scale in GitHub)
- **Optimize:** Compress before committing
  - Use tools like TinyPNG, ImageOptim
- **Accessibility:** Add alt text in markdown
